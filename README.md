# Research-Hub

Academic personal website for **Junteng Liu**, powered by the [Academic Pages](https://github.com/academicpages/academicpages.github.io) template and hosted on GitHub Pages.

## About

This site showcases the research, publications, and academic profile of Junteng Liu — a first-year PhD candidate at HKUST NLP Group, focusing on natural language processing, LLM reasoning, hallucination in vision-language models, and LLM truthfulness.

## Local Development

```bash
bundle install
bundle exec jekyll serve
```

Visit `http://localhost:4000/Research-Hub/` to preview.
